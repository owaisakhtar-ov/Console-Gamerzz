# Console-Gamerzz
## A Console Based Game
![alt text](https://i.ibb.co/j3L37RX/Untitled.jpg)

A Console Based Game including 
### Tic Tac Toe
- 2 Players 
- vs Computer
  
### Sudoku
  with 3 difficulty levels
 - Easy 6 x 9
 - Medium 9 x 9
 - Hard 9 x 9
  
### Hangman
  Word Guessing game
